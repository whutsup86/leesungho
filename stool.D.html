stool.D

Development of LED Feedback, Aug 2019 - Nov 2019

Keywords: Stool, Cycle, LED, Generator, Furniture, Small house



stool.D is a combination stool and cycling machine designed by SangJin, won the Red Dot Award 2018 and invited Dubai Design Week 2018. I engaged in development activity as an electronic system engineer for two months. When the user pedals, LED provide visual feedback. The light rotates at the same rate as the pedaling speed and provides the user with pleasure and motivation.

- Designed electronic system : STM32, Reed Switch, Generator, IR sensor, Super Capacitor
- Programming with C, C++
- Designed PCB
- Tested PCB & PCA
- Completed LED feedback
