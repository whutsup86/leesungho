AnyBaro

IoT Exercise Equipment, Jul 2018 - Jul 2019

Keywords: IoT Device, Exercise Equipment

Patent: KR3010164160000(2019.07.19)


AnyBaro is an IoT exercise equipment. The main functions are as follows. First, it provides the balance of body by detecting the pressure of feet. Second, it helps user to exercise the milking action of leg muscles through user customized repetitive exercise
